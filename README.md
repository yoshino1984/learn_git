# Python 学习示例项目

这是一个用于学习 Python 的简单示例项目。

## 项目结构

```
first_project_for_cc/
├── main.py          # 主程序入口
├── utils.py         # 工具函数模块
├── requirements.txt # 项目依赖
└── README.md        # 项目说明
```

## 使用方法

### 运行主程序

```bash
python main.py
```

### 安装依赖

如果需要安装依赖包：

```bash
pip install -r requirements.txt
```

## 学习要点

1. **模块化编程** - 将代码拆分为不同的模块
2. **函数定义** - 使用 `def` 定义函数
3. **文档字符串** - 使用 `"""..."""` 编写函数说明
4. **条件判断** - 使用 `if/elif/else` 进行逻辑判断
5. **异常处理** - 使用 `raise` 抛出异常
